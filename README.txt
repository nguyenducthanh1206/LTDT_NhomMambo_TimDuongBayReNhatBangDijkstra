ĐỒ ÁN MÔN HỌC: LÝ THUYẾT ĐỒ THỊ

Tên đề tài
Mô phỏng thuật toán Dijkstra trong việc tìm chuyến bay rẻ nhất giữa các thành phố lớn trên thế giới

1. Giới thiệu
Dự án này được thực hiện trong khuôn khổ đồ án môn học Lý thuyết đồ thị. Mục tiêu của đồ án là nghiên cứu, cài đặt và mô phỏng thuật toán Dijkstra nhằm giải quyết bài toán thực tế: tìm lộ trình bay có tổng chi phí thấp nhất giữa các thành phố lớn trên thế giới.

Bài toán được mô hình hóa dưới dạng đồ thị có trọng số, trong đó mỗi đỉnh biểu diễn một thành phố, mỗi cạnh biểu diễn một chuyến bay và trọng số của cạnh là chi phí vé máy bay.

2. Mục đích
Dự án được xây dựng phục vụ cho mục đích học tập và nghiên cứu trong khuôn khổ đồ án môn học, không có mục đích thương mại. Thông qua đồ án, sinh viên hiểu rõ hơn về thuật toán Dijkstra và khả năng ứng dụng của thuật toán trong các bài toán thực tế.

3. Nội dung thực hiện
- Cài đặt và mô phỏng thuật toán Dijkstra  
- Mô hình hóa bài toán tìm chuyến bay rẻ nhất dưới dạng đồ thị  
- Xây dựng và thực hiện các test case minh họa  
- Phân tích và đánh giá kết quả của thuật toán  

4. Công nghệ sử dụng
- Ngôn ngữ lập trình: Node.js  
- Môi trường phát triển: Visual Studio Code  
- Công cụ quản lý mã nguồn: Git, GitHub  

5. Hướng dẫn cài đặt và chạy chương trình
Bước 1: Clone repository  
git clone https://github.com/nguyenducthanh1206/LTDT_NhomMambo_TimDuongBayReNhatBangDijkstra.git

Bước 2: Cài đặt các thư viện cần thiết  
npm install

Bước 3: Chạy chương trình  
npm start

6. Lưu ý
Mã nguồn không chứa phần mềm độc hại, không bao gồm file thực thi hoặc chương trình cài đặt. Thư mục node_modules không được đưa lên GitHub theo đúng chuẩn phát triển phần mềm và có thể được cài đặt lại bằng lệnh npm install. Dự án chỉ sử dụng cho mục đích học tập.
